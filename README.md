# Tocaria & Terrenos S.A.

Site especifico para vendas de propriedades e imóveis na região do condado.

---

## Fase 1 concluída
O projeto será separado em 2 fases. A primeira é constituida pelo corpo do projeto, onde vai ficar disponível os campos mais simples e a idéia do projeto final.
A fase será focada em transformar a estrutura estática do projeto em um site funcional, utilizando JS para criar dinamismo e concluir os requisitos do trabalho.

### Páginas até o momento
* `index.html` (Home)
* `páginas/sobre.html`
* `páginas/produtos.html`
* `páginas/servicos.html`
* `páginas/contatos.html`

### Linguagens utilizadas
* **HTML5**
* **CSS3** (Compilado via SCSS)
* **Bootstrap 5.3** (Framework CSS/Responsividade)
* **JavaScript** (Configuração inicial para validação Front-End futura)
